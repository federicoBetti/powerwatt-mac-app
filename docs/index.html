PowerWatt

A lightweight macOS menu bar app that shows your battery power usage (IN/OUT watts) in real time.

Download PowerWatt (DMG) Download ZIP

These links always point to the latest GitHub Release.

Install

  • Open the downloaded .dmg and drag PowerWatt.app to Applications.
  • On first launch, if macOS blocks the app, right-click PowerWatt.app → Open.
  • PowerWatt runs in the menu bar (no Dock icon).

Privacy

Optional anonymous usage statistics are off by default and can be enabled/disabled anytime in Preferences.
