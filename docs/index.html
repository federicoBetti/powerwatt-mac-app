Features

A focused tool that does one thing well: show power direction + watts, always visible.

Real-time watts

Battery power usage (IN/OUT)

Shows charging vs discharging watts using system battery readings.

Display modes

Net-only or separate IN/OUT

Pick “Net Only (IN−OUT)” or “Separate IN and OUT” depending on your workflow.

Readable at a glance

Label styles + colors

Choose icon+watts, watts-only, or prefix+watts—optionally color-coded.

Control

Refresh rate

Adjust polling frequency (1–60 seconds) to balance responsiveness and noise.

Stability

Smoothing window

Average readings over a short window (0–30 seconds) to reduce flicker.

System integration

Open at login + updates

Start automatically and install updates via Sparkle from the official appcast.

Tip:

If you’re profiling battery drain, try enabling “Separate IN and OUT”, then set a short refresh interval and a small smoothing window.

Install

Quick install

  1. Download PowerWatt.dmg.
  2. Open the DMG and drag PowerWatt.app into Applications.
  3. Launch PowerWatt. It appears in the menu bar.

First launch notes

  • If macOS blocks the app, right-click PowerWatt.app → Open.
  • Preferences are available from the menu bar dropdown.
  • For best “Open at Login” behavior, keep the app in /Applications.

Privacy-first by design

Off by default

Anonymous usage stats are optional. On first launch, PowerWatt asks if you want to share anonymous telemetry to help prioritize improvements.

You can enable/disable it at any time in Preferences.

FAQ

How does PowerWatt calculate watts?

It reads battery voltage (mV) and current (mA) from macOS and computes watts as (voltage × current) / 1,000,000.

Why do values sometimes jump?

Short spikes are normal when your Mac changes CPU/GPU load. Use the smoothing window to average readings and reduce flicker.

Does it work on desktops?

PowerWatt reads battery data, so it’s intended for Macs with a battery.

Where can I report issues?

Please open an issue on GitHub with your macOS version and what you expected to see.
