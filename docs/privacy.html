Privacy

PowerWatt can optionally send anonymous usage statistics to help improve the app. This is off by default.

Consent

  • Telemetry is disabled by default.
  • On first launch, PowerWatt asks if you want to share anonymous usage stats.
  • You can enable/disable telemetry at any time in Preferences.

What we collect (minimal, anonymous)

When enabled, PowerWatt may send events such as:

  • app_first_launch
  • app_launch
  • view_opened (e.g. Preferences)
  • setting_changed (setting name and a bucketed value)
  • feature_used (feature name)
  • error_nonfatal (error domain/code and short context)

Events include a random install_id and basic app/device context:

  • App version and build number
  • macOS version
  • Locale (language/region)

What we do NOT collect

  • No name, email address, or account identifiers
  • No macOS username, computer name, or serial number
  • No precise location data
  • No session replay

Processor / hosting

Telemetry is processed by PostHog Cloud (EU region). The endpoint used is https://eu.i.posthog.com.

How to opt out

Open PowerWatt Preferences and turn off “Share anonymous usage stats”. When disabled, PowerWatt does not initialize telemetry and does not send events.
